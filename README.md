# README

This is a roguelike made in Rust, as a tutorial.

## Usage

You must have rust installed, with cargo in your path.

1. `git clone <https://github.com/pabrams/rustcod.git>`
1. `cd rustcod`
1. `cargo build`
1. `cargo run`
